Git and GitHub Setup

Create a GitHub Account

You should already be added to the GitHub Classroom for this course. If not, I suggest reaching out to me first. Otherwise, if you have your own reasons, you can create an account at github.com.

Install Git

  • Windows: Download Git for Windows. This includes Git Bash, which is a great terminal for beginners.
  • Mac: Install Xcode Command Line Tools: xcode-select --install
  • Linux: Use your package manager, e.g., sudo apt install git or sudo dnf install git

Configure Git

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

Set Up SSH Authentication

Once you've generated your SSH keypair (see this guide), upload your public key to GitHub at github.com/settings/ssh/new.

Test SSH Connection

Inside of the terminal, run the following command: 
ssh -T git@github.com

If successful, you'll see a greeting message.
