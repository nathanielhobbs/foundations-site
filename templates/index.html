4.2 odd_indices(L): elements at odd indices (1, 3, 5, ...) where the value is even.

def odd_indices(L):
    """Return elements at odd indices whose values are even."""
    result = []
    for i in range(1, len(L), 2):
        if L[i] % 2 == 0:
            result.append(L[i])
    return result

# or, as a comprehension:
# return [L[i] for i in range(1, len(L), 2) if L[i] % 2 == 0]

4.3 Short explanation: sys.argv vs argparse.

  • sys.argv is just the raw list of strings from the command line (including the script name). You manually check lengths, positions, and convert types.
  • argparse lets you declare positional arguments and options (like --verbose, --year), with types, defaults, and automatic help/error messages.
  • Example where argparse is clearly better: a script that needs multiple optional flags such as --input, --output, --limit N, and --verbose. With argparse you avoid manual parsing of sys.argv and get built-in usage text and validation.

4.4 merge_counts(d1, d2): combine counts, summing overlapping keys.

def merge_counts(d1, d2):
    """Return a new dict whose keys are from both d1 and d2.
    If a key appears in both, sum the values.
    """
    result = dict(d1)  # copy so we don't modify d1
    for key, value in d2.items():
        if key in result:
            result[key] += value
        else:
            result[key] = value
    return result
