Generating SSH Keypairs

What is an SSH Keypair?

An SSH keypair allows you to authenticate securely with GitHub and other services without entering your password each time. It consists of a private key (kept secret) and a public key (shared).

Generate a Keypair

Open your terminal (or Git Bash on Windows) and run:

ssh-keygen -t ed25519 -C "you@example.com"

Press Enter to accept the default save location. When prompted for a passphrase, just press Enter to skip.

Upload Your Public Key to GitHub

cat ~/.ssh/id_ed25519.pub

Copy the output, then visit github.com/settings/ssh/new and paste it in.

Verify Setup

ssh -T git@github.com
